# 内网 GM 助手（v1.8.2）

浏览器插件形态的 GM 场景化操作加速器。  
定位是“命令工作台 + 智能参数选择”，不替代后台原生命令联想，不自动发送命令。

## 核心能力

- 悬浮球 + 侧边栏（支持拖动、位置记忆）
- 侧边栏宽度支持拖拽拉伸（最小宽度 420，自动记忆上次宽度）
- 左侧导轨双页签：`命令库`、`设置`
- 命令工作台：参数表单、结果预览、生成/复制/填入/追加
- SmartPicker：搜索、多选、全选/反选/清空、批量一项一条生成
- `send_template_mail` 多附件构建器：道具可搜索多选、加入附件列表后统一生成 `itemid/num/bind;...`
- 外部词典源：绑定 `Item.xlsx` / `Email.xlsx` / `Task.xlsx`，手动刷新
- 自定义命令模板：保存多行命令，一键填入后台输入框
- 多皮肤系统：`Default` / `Linear` / `Sentry` / `Notion`，支持设置页一键切换并持久化记忆

- 修复：分组折叠/展开、切换 UID 操作补全 toast 反馈。

## v1.7.0 版本改动（交互体验优化版）

- 新增（键盘流）：打开侧边栏即聚焦搜索框；搜索框回车直达首条推荐命令；`Esc` 在 picker 有内容时清空、否则关闭侧边栏；picker 输入框补「↑↓ 选择 · Enter 勾选」提示。
- 新增（操作反馈）：右下角 toast 浮层，生成/复制/填入/加入附件等操作即时提示，成功 ~3s 淡出、失败久留;不随主面板重渲染被清掉。
- 优化（防误触 + 确认）：悬浮球拖拽阈值 4px → 7px,减少手抖误判;危险命令改为结果区内联二次确认条（确认/取消），并移除「确认一次后永久免确认」，每次填入都需确认。
- 优化（邮件多附件）：道具勾选即加入附件列表（去掉「加入附件列表」二步），取消勾选即移除；附件列表头新增「数量 + 绑定 + 应用到全部」批量编辑。
- 优化（搜索推荐面板）：合并原本重复的两排操作 chip，改为「勾选道具（可选）→ 选择操作」单一流向；操作 chip 始终可点，勾了道具即带入、未勾直接打开命令。
- 修复：从搜索推荐预填道具到 `send_template_mail` 时，多选道具现可正确生成完整附件行（此前只会带入第一个）。
- 重构（根治失焦/卡顿）：高频输入（搜索框、picker 搜索、picker 上下键）从「整树重建 + 焦点抢救」改为局部刷新，杜绝「输入一字失焦」与中英混输丢字，菜单实时筛选更顺滑；离散动作仍保留整树渲染并带安全回退。

## 版本更新提醒（GitHub Release）

- 版本源：`https://api.github.com/repos/Kayungko/GM-Assistant/releases/latest`
- 检查策略：
  - 自动：每 24 小时最多检测一次
  - 手动：设置页“立即检查更新”
- 提醒方式：检测到新版本时弹窗提醒，支持“稍后提醒”

## 版本对齐规则

- `manifest.json` 中 `version` 作为插件当前版本（当前为 `1.8.2`）
- GitHub 发布规范：
  - Tag：`vX.Y.Z`
  - Manifest：`X.Y.Z`
- 示例：发布 `v1.8.2` 时，`manifest.version` 必须是 `1.8.2`

## 安装

1. 打开扩展管理页  
   - Chrome: `chrome://extensions/`  
   - Edge: `edge://extensions/`
2. 开启“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择本目录：`C:\Users\admin\Desktop\内网GM助手`

## 执行边界

- 插件只做“生成/复制/填入/追加”
- 不自动点击发送
