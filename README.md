# 内网 GM 助手（V1.2）

浏览器插件形态的 GM 操作加速器，定位是“命令工作台 + 智能参数选择”，不替代后台原生命令联想，不自动发送。

## 主要能力

- 悬浮球 + 侧边栏，支持拖动定位与位置记忆
- 首页双栏信息架构：搜索、当前 UID、最近/收藏、高频分组入口
- 命令工作台：参数表单、预设、结果预览、复制/填入/追加
- 智能参数选择器（SmartPicker）：
  - 模糊搜索
  - 多选/全选/反选/清空
  - 键盘上下选择与回车确认
  - 语义别名词命中（如“补偿邮件”“刷物品”）
- 批量生成规则：一项一条命令，超过 200 条会二次确认
- 自动识别后台输入框，优先当前聚焦命令框
- 危险命令首次执行确认

## 数据与目录

- 内置词典按业务分组存放：
  - `data/catalogs/items/*.json`
  - `data/catalogs/mail/*.json`
  - `data/catalogs/tasks/*.json`
  - `data/catalogs/common/*.json`
- 统一入口：`data/catalogs/registry.json`
- 运行时只通过 registry 装载分组包（支持分组启停与 schema 校验）

## 导入

- 支持手动导入 `.xlsx` / `.json`
- 导入后采用“单活快照”策略：覆盖上一份导入词典
- 当前重点支持：
  - `Item.xlsx`（物品索引）
  - `Email.xlsx`（邮件模板索引 + 附件规则扩展）

## 安装

1. 打开浏览器扩展页（Chrome: `chrome://extensions/`，Edge: `edge://extensions/`）
2. 开启开发者模式
3. 选择“加载已解压扩展程序”
4. 目录指向：`C:\Users\admin\Desktop\内网GM助手`

## 执行边界

- 插件只负责“生成 / 复制 / 填入 / 追加”
- 不自动点击发送
